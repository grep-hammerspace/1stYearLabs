## Data Structures and Algorithms

This repo consists of code used to practice coding for year 1 labs. It is built with Java 21 and Maven.

This repo contains class stubs (what you fill out when doing the labs) for the following data structures and algorithms:

### Data Structures
Array
Stacks
Queues
Linked Lists
HashMaps
Binary Search Trees

### Algorithms
Binary Search
Bubble Sort
Merge Sort
Recursion
Depth First Search
Breadth First Search

### Structure 

This repo is laid out in the following way:

```
src/
    main/
        java/
            com/
                example/
                    data-structures-and-algorithms/
                        <code for labs>
                    answers/
                        <code for answers>
    test/
        java/
            com/
                example/
                    data-structures-and-algorithms/
                        <code for tests>
```

### Running Tests
When you're ready to do the labs, swap the import in any test file from `org.example.answers.X` to `org.example.dsa.X`
